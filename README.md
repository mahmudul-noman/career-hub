# Project: **Job Station 7**

The website help job seekers find job opportunities that match their skills and experience and streamline the job application process.


## Deploy Link
https://job-station7.netlify.app/


## Features

- Job Category List
- Featured Job 
- Job Details Page
- Applied Jobs Page
- Filter Applied Jobs by (Remote or Onsite)
- Blog Page
- Responsive design
- High-Speed Performance
- Great Menu for Small Devices
- Smooth Scroll
- Statistics Page
- Cross platform


## Run Locally

Clone the project

```bash
  git clone project_url
```

Go to the project directory

```bash
  cd project_name
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
